# tmux

[返回目录](/linux/README.md)

---

- 安装tmux：`yum install tmux -y`
- 开启一个session：`tmux`
- 恢复到上次启动的session：`tmux a`
- 查看全部session：`tmux ls`
- 删除指定session：`tmux kill-session -t session名`
- 横向分割窗口：`Ctrl+b "`
- 竖向分割窗口：`Ctrl+b %`
- 移动到指定方向的窗口：`Ctrl+b 方向键`
- 关闭当前窗口：`Ctrl+d`
